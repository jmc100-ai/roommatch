Recommendation — simplify bundled sheet

  • Default to Compact layout — ~40% less scroll than full desktop parity; same chip, all features preserved.
  • Budget: 2×2 pills (Any / $250 / $400 / Luxury) + “Custom range” link — drop per-card icons and subtitles.
  • Flex + dates: one-line toggle and inline “Add dates” text — not boxed banners (show only when relevant).
  • Must-haves: wrap chips, no photo rows — hints live in wizard; sheet is for quick toggles.
  • Extras: “Anything else?” textarea open by default; Available only stays as one slim row at bottom.
  • Cut duplicate copy: drop sheet subtitle + section icon circles; footer already summarizes selection.
  • Keep Full layout only if you want literal desktop markup reuse — otherwise Compact is the ship target.
