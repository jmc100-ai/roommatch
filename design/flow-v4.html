v4 — Changes & rationale

Six targeted fixes from the latest feedback round. Same core flow as v3, sharpened.

Brand: Hotel Vibe Search All content centered (max 1180px) Story bar: 72px, larger chips/text Nbhd cards: hover rich detail Describe-it: prominent search bar Results cmd bar: 68px, 15px text

Brand: Hotel Vibe Search

  • Logo: "Hotel Vibe Search" replaces "travelboop"
  • "Vibe" used throughout: "neighbourhood vibe", "room vibe", "find your vibe"
  • Story bar labels: "Neighbourhood Vibe" and "Room Vibe"
  • Results: "ranked by vibe match" instead of "room match"
  • Search placeholder: "Refine your vibe —…"

Centering (desktop)

  • --max: 1180px applied to all content sections via .chapter-inner
  • Story bar inner, cmd bar, results body — all centered
  • App header — centered with max-width wrapper
  • Gutters: clamp(20px, 4vw, 48px) — breathes on wide screens
  • Works like Booking.com / Airbnb content columns

Story bar — bigger & more obvious

  • Height: 52px → 72px
  • Value font: 13px → 15px, font-weight 500 when set
  • Chip padding: 5px 10px → 8px 14px
  • Dates CTA: 12px padding → 10px 20px, 13px text
  • Border-bottom bumped to 2px for stronger separation
  • Subtle amber glow on box-shadow when visible

Neighbourhood cards — hover detail

  • Cards larger: minmax(220px) → minmax(280px), aspect-ratio 4/3
  • Hover: second gradient overlay fades in (covers 50% of card)
  • Hover: .nbhd-detail slides up — 2-sentence description, best-for pills, hotel count, pick button
  • CSS-only transition, no JS needed for hover
  • Background subtly scale(1.04) on hover for depth

Describe-it text box

  • Moved out of a small inline row into its own "describe-section"
  • Same amber-glow styling as the city search box
  • Font size 16px (was 13px), icon prefix, full-width
  • Max-width 820px — spans most of the column
  • Label copy: "Or describe your perfect room — AI understands anything"
  • Button: "Find rooms →" (vs the old tiny "Use this →")

Results command bar

  • Height: 52px → 68px
  • Value font: 13px → 15px, font-weight 500
  • Segment padding: 10px 16px → 14px 20px
  • Border-bottom: 1px → 2px for strong visual anchor
  • Box-shadow stronger (24px blur)
  • Segment label "Neighbourhood Vibe" reinforces brand language
